# miniguia-data-driven

## Contexto e Objetivos

Este projeto foi desenvolvido durante meu processo de transição de carreira para a área de tecnologia, com foco em desenvolvimento de software e análise de dados.
Diante da crescente importância da tomada de decisão baseada em dados no mercado, surgiu a necessidade de compreender não apenas conceitos técnicos, mas também desenvolver uma mentalidade orientada a dados (Data Driven).
O uso de Inteligência Artificial, por meio do NotebookLM, foi incorporado como ferramenta de apoio ao aprendizado, permitindo explorar o conteúdo de forma ativa, estruturada e crítica.

### Objetivos de estudo:

- Compreender os fundamentos da Ciência de Dados
- Desenvolver pensamento analítico e orientado a dados
- Aprender a estruturar conhecimento a partir de múltiplas fontes
- Melhorar a qualidade das interações com IA (engenharia de prompts)
- Criar um material de estudo reutilizável e aplicado à prática profissional
